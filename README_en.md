Programming vacancies compare

The project compares programmer vacancies from Habr Career and SuperJob APIs.
It calculates average salaries for various programming languages in Moscow.

How to install

1. Clone the repository
2. Create virtual environment: `python -m venv .venv`
3. Activate it: `.venv\Scripts\activate` (Windows) or `source .venv/bin/activate` (Linux/macOS)
4. Install dependencies: `pip install -r requirements.txt`
5. Create `.env` file with `SUPERJOB_APP_ID=your_api_key`
6. Run: `python parser.py`
